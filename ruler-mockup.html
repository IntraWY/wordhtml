Ruler — ปรับให้เหมือน Microsoft Word

Mockup สำหรับเลือกว่าจะทำจุดไหนบ้าง · ลองจับลากได้จริงทุกอัน (ไม่กระทบโปรเจคจริง)

ระบบ ruler ปัจจุบัน ใกล้เคียง Word มากแล้ว — 4 การ์ดข้างล่างคือสิ่งที่ ยัง ต่างจาก Word

1 ซม. = 37.8px (A4 96 DPI) 🟢 = ทำง่าย/เสี่ยงน้อย 🔴 = งานใหญ่/เสี่ยง ลากสามเหลี่ยม/สี่เหลี่ยม/ตัวอักษรบนไม้บรรทัดได้

3Right-indent marker บนไม้บรรทัด / Right indent handle

Impact: กลางEffort: ต่ำ 🟢

★ แนะนำ — Word มีสามเหลี่ยมย่อหน้าฝั่งขวาให้ลากบนไม้บรรทัด แต่ของเรายังต้องตั้งผ่านกล่อง "ย่อหน้า" เท่านั้น ค่า marginRight มีอยู่แล้วในโค้ด แค่เพิ่ม handle ลากได้ · ลองลากสามเหลี่ยม 🔵 ฝั่งขวา

ตัวอย่างย่อหน้า — ลากสามเหลี่ยมขวาเข้ามาแล้วข้อความฝั่งขวาจะหดตามเหมือนใน Microsoft Word ทุกประการ ทดสอบได้โดยลากเข้า–ออก ข้อความจะ reflow ทันที และตัวเลขด้านล่างจะอัปเดตค่าระยะเป็นเซนติเมตร เพื่อให้เห็นว่าฟีเจอร์นี้พร้อมต่อยอดลงโปรเจคจริงได้เลย

1Tab stop จัดข้อความได้จริง / Per-position tab stops

Impact: สูงEffort: สูง 🔴

จุดต่างที่สำคัญสุด — ตอนนี้ไม้บรรทัดโชว์ stop หลายจุด แต่ตอน render ใช้แค่ stop แรก เป็นระยะ tab-size เท่า ๆ กัน → tab ที่ 1.5/7/12 ซม. จึงไม่ลงตรงตำแหน่งจริง · กดสลับโหมดเพื่อเทียบ และลากตัว L เพื่อย้าย stop

2ชนิด Tab + ตัวเลือกมุม / Tab types & corner selector

Impact: กลางEffort: สูง 🔴

Word มี tab ซ้าย / กลาง / ขวา / ทศนิยม / เส้น (L C R D |) เลือกจากกล่องมุมซ้าย แล้วคลิกไม้บรรทัดเพื่อวาง · ของเรามีแต่ซ้าย · คลิกกล่องมุมเพื่อสลับชนิด แล้วคลิกบนไม้บรรทัดเพื่อวาง stop (ต่อยอดจากข้อ 1)

L ชนิดปัจจุบัน: ซ้าย (Left) — คลิกไม้บรรทัดเพื่อวาง

4Double-click + สี่เหลี่ยม Left-indent / Dbl-click dialog & combined marker

Impact: เล็กEffort: ต่ำ 🟢

Word: ดับเบิลคลิกไม้บรรทัดเปิด Page Setup/ย่อหน้า และมีสี่เหลี่ยมใต้สามเหลี่ยมที่ลากแล้วเลื่อน first-line + left พร้อมกัน · ลากสี่เหลี่ยม ◼ / ดับเบิลคลิกช่องขอบหรือช่องย่อหน้า

ลากสี่เหลี่ยม ◼ ด้านล่าง → สามเหลี่ยมบน (first-line) และซ้าย (left) จะขยับไปพร้อมกัน เหมือน Word ส่วนดับเบิลคลิกบริเวณช่องขอบกระดาษ (สีลายทาง) จะเปิดกล่อง Page Setup จำลอง และดับเบิลคลิกในช่องย่อหน้าจะเปิดกล่อง "ย่อหน้า" จำลอง
เคล็ดลับ: ดับเบิลคลิก ช่องลายทาง = Page Setup · ดับเบิลคลิก กลางไม้บรรทัด = ย่อหน้า
